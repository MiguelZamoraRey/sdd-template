# Código fuente

Este directorio contiene el código generado a partir de las specs aprobadas en `specs/`.

## Convenciones

- Cada módulo debe referenciar la spec de la que proviene en un comentario al inicio del archivo.
- No crear código sin una spec `approved` correspondiente.
- Si necesitas lógica no cubierta por ninguna spec, crea o actualiza la spec primero.

## Ejemplo de referencia en código

```js
// Spec: specs/features/user-authentication.md (FEAT-001)
// AC-1: Autenticación con credenciales válidas
```
